农业--过垦过牧、过度砍伐森林对环境造成的影响

  1. 土地荒漠化
  2. 水土流失
  3. 土地肥力下降
  4. 草场退化
  5. 破坏生态平衡
  6. ...

农业-梯田的作用

  1. 保持水土，缓解水土流失
  2. 增加耕地面积，提高土地利用率
  3. 发展旅游资源
  4. 改善小范围生态环境

环境保护--热带雨林的作用

  1. 提供新鲜空气，制造氧气
  2. 调节气候
  3. 涵养水源
  4. 维护生物多样性
  5. 保护土壤

环境保护---如何保持水土

  1. 植树种草
  2. 缓坡修建梯田（陡坡别修）
  3. 退耕/牧还草
  4. 打淤地坝

降水---降水的影响因素

  1. 距海近降水多，反之降水少
  2. 迎风坡降水多，背风坡降水少
  3. 回归线两侧，大陆东岸降水多，大陆西岸降水少
  4. 高山会阻挡湿润气流（e.g.塔里木盆地四面高山环绕，呈环形封闭状，降水少）

降水-影响我国降水的主要因素

  • 海陆位置
  • 地形因素
  • 纬度位置

水源方面--降水时空分布不均的解决方法

  • 时：兴建水库
  • 空：跨流域调水

水源方面--建水库的作用

  1. 蓄水
  2. 防洪
  3. 水产养殖
  4. 发电

其他人类活动

地区发展工业的条件

  1. 矿产资源丰富
  2. 交通便利
  3. 劳动力资源丰富，劳动力成本低
  4. 科技发达

分析当地民居特点

  1. 降水量大的地方一般屋顶较陡
  2. 降水量少/干旱的地方一般屋顶是平的
  3. 草原等发展畜牧业的区域使用蒙古包---游牧需逐水草而居，蒙古包易拆装方便移动

地区发展旅游业的条件

  1. 丰富的旅游资源
  2. 完善的基础设施建设
  3. 优质的服务
  4. 便利的交通

交通运输方面-管道运输的优点

  1. 连续运输
  2. 不受天气影响
  3. 安全性高
  4. 线路短

分析当地自然环境对村庄/城市/聚落分布的影响

  1. 水源：水源是否充足---一般城市/村庄/聚落会分布在河流沿岸，绿洲附近，山麓地带
  2. 土壤肥沃：适宜耕种
  3. 自然资源丰富：例如矿产资源、油气资源。适于发展工业，带动经济发展
  4. 交通：沿海、沿江交通发达
  5. 地形：平坦

河流对地区发展的影响

  1. 提供工业用水、生活用水、农业用水
  2. 若地势起伏较大（e.g.第一第二阶梯交界处）可发展水能资源
  3. 航运
  4. 冲击形成平原
  5. 淡水渔业

河流--发展水能资源

  • 水能资源属于可再生能源
  • 利用水能资源--建水电站
  • 水电站建在落差大的地方（e.g.我国第一级阶梯和第二级阶梯的交界处），因为这样水流湍急

影响河流航运价值的因素

  1. 支流数量
  2. 长度（通航里程）
  3. 无结冰期，全年通航
  4. 河流流量大
